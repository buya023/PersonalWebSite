Binary Search is one of the most efficient and elegant algorithms in computer science. It helps you find an element in a sorted list quickly ‚Äî in just O(log n) time. Let‚Äôs explore how it works step by step in a simple way üëá

1. What Is Binary Search?

Binary search is a divide and conquer algorithm. It repeatedly divides the search space in half until the target value is found (or determined not to exist).

Imagine you‚Äôre searching for a word in a dictionary ‚Äî you don‚Äôt check each word one by one. Instead, you open the middle, compare, and decide whether to go left or right. That‚Äôs exactly what binary search does.

2. When Can You Use It?

  • The data must be sorted (ascending or descending).
  • You must have random access (like in an array or list).

3. How It Works (Step-by-Step)

Let‚Äôs take an example:

nums = [1, 3, 5, 7, 9, 11]
target = 7

Steps:

  1. Start with two pointers: left = 0, right = len(nums) - 1
  2. Find the middle index: mid = (left + right) // 2
  3. Compare nums[mid] with target:
    • If equal ‚Üí found it!
    • If smaller ‚Üí move left to mid + 1
    • If larger ‚Üí move right to mid - 1
  4. Repeat until left > right (meaning not found).

4. Python Example

def binary_search(nums, target):
    left, right = 0, len(nums) - 1

    while left <= right:
        mid = (left + right) // 2
        if nums[mid] == target:
            return mid  # found
        elif nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1

    return -1  # not found

print(binary_search([1, 3, 5, 7, 9, 11], 7))  # Output: 3

6. Time Complexity

  • Best case: O(1) (found in first check)
  • Worst case: O(log n)
  • Space: O(1) (iterative)

7. Common Binary Search Variants

  • Find first or last occurrence of an element
  • Find smallest element greater than or equal to target
  • Search in rotated sorted array
  • Find square root, peak element, insert position
