bisect is a Python module that helps you perform fast binary search on sorted lists. It lets you insert, locate, or maintain elements efficiently without manually looping.

ğŸ’¡ Why use bisect?

  • Built-in binary search (no loops required)
  • Runs in O(log n) time
  • Keeps lists sorted automatically
  • Perfect for ranking systems, leaderboards, or cumulative data

ğŸ§© Example: Searching for a Value

import bisect

nums = [1, 2, 4, 4, 5]
x = 4

i = bisect.bisect_left(nums, x)
if i < len(nums) and nums[i] == x:
    print(f"{x} found at index {i}")
else:
    print(f"{x} not found")

# Output: 4 found at index 2

ğŸ” Example: Counting Duplicates Efficiently

import bisect

nums = [1, 2, 4, 4, 4, 5]
left = bisect.bisect_left(nums, 4)
right = bisect.bisect_right(nums, 4)
count = right - left

print("Number of 4s:", count)
# Output: Number of 4s: 3

âš”ï¸ Real Use Case: Find Common Element in Two Sorted Arrays

import bisect

def getCommon(nums1, nums2):
    if len(nums1) > len(nums2):
        nums1, nums2 = nums2, nums1

    for num in nums1:
        i = bisect.bisect_left(nums2, num)
        if i < len(nums2) and nums2[i] == num:
            return num
    return -1

print(getCommon([1,2,3,6], [2,4,5,6]))
# Output: 2
