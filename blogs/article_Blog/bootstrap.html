Bootstrap Cheat Sheet: Learn Fast

By Buya • September 2025

Bootstrap is a front-end framework to build responsive websites quickly. This cheat sheet covers all essential classes, components, and utilities in one place.

1. Grid System (Layout)

<div class="container"> <!-- or container-fluid -->
  <div class="row">
    <div class="col-md-4">Column 1</div>
    <div class="col-md-4">Column 2</div>
    <div class="col-md-4">Column 3</div>
  </div>
</div>
  • .container → Wrap content
  • .row → Horizontal row
  • .col-* → Columns, responsive (sm, md, lg)

2. Typography

<p class="text-center fw-bold fst-italic text-primary">Text Example</p>
<p class="text-end text-danger">Right-aligned</p>
  • .text-* → Align text (start, center, end)
  • .fw-* → Font weight (bold, light)
  • .fst-italic → Italic
  • .text-* → Colors (primary, danger, success)

3. Buttons

<button class="btn btn-primary btn-lg">Primary</button>
<button class="btn btn-outline-success btn-sm">Outline</button>
  • .btn → Base button
  • Colors: primary, secondary, success, danger, warning
  • Sizes: .btn-lg, .btn-sm

4. Forms

<input type="text" class="form-control" placeholder="Name">
<select class="form-select">
  <option>Option 1</option>
</select>
<div class="form-check">
  <input type="checkbox" class="form-check-input">
</div>
  • .form-control → Input
  • .form-select → Dropdown
  • .form-check → Checkbox/Radio
  • Validation: .is-valid, .is-invalid

5. Utilities

<div class="d-flex justify-content-between align-items-center border p-3 m-3 rounded">
  Flex Box Items
</div>
  • .m-* / .p-* → Margin / Padding
  • .d-flex → Flex
  • .justify-content-* → Horizontal alignment
  • .align-items-* → Vertical alignment
  • .border / .rounded → Borders & rounded corners

6. Navbar

<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <a class="navbar-brand" href="#">Logo</a>
  <button class="navbar-toggler" data-bs-toggle="collapse" data-bs-target="#menu">
    <span class="navbar-toggler-icon"></span>
  </button>
  <div class="collapse navbar-collapse" id="menu">
    <ul class="navbar-nav ms-auto">
      <li class="nav-item"><a class="nav-link active" href="#">Home</a></li>
    </ul>
  </div>
</nav>

7. Cards

<div class="card" style="width: 18rem;">
  <div class="card-body">
    <h5 class="card-title">Title</h5>
    <p class="card-text">Text content</p>
    <a href="#" class="btn btn-primary">Button</a>
  </div>
</div>

8. Alerts

<div class="alert alert-success alert-dismissible fade show">
  Success alert!
  <button class="btn-close" data-bs-dismiss="alert"></button>
</div>

9. Badges & Pills

<span class="badge bg-primary">New</span>
<span class="badge bg-danger rounded-pill">Hot</span>

10. Images

<img src="https://via.placeholder.com/150" class="img-fluid rounded">
<img src="https://via.placeholder.com/100" class="img-thumbnail rounded-circle">

11. Tables

<table class="table table-striped table-bordered table-hover table-sm">
  <tr><th>#</th><th>Name</th></tr>
  <tr><td>1</td><td>John</td></tr>
</table>

12. Modals

<button class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#modal">Open Modal</button>
<div class="modal fade" id="modal">
  <div class="modal-dialog">
    <div class="modal-content p-3">Modal Content</div>
  </div>
</div>
