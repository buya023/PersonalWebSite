Twig is a powerful templating engine in Symfony. Here are the most used features that cover 90% of daily work with Twig.

1. Variables & Filters (Output)

{{ variable }}             {# Print variable #}
{{ user.name|upper }}      {# Apply filter: BUYA #}
{{ post.date|date("Y-m-d") }}  {# Format date #}
{{ items|length }}         {# Count items #}
{{ var|default('N/A') }}   {# Fallback if var is null #}

2. Conditionals

{% if user %}
    Hello {{ user.name }}
{% else %}
    Guest
{% endif %}

3. Loops

{% for product in products %}
    {{ loop.index }}. {{ product.name }}
{% else %}
    No products found.
{% endfor %}

Loop helpers:

  • loop.index â†’ 1-based index
  • loop.first, loop.last
  • loop.revindex â†’ reverse index

4. Template Inheritance (Layouts)

Base layout:

<html>
  <body>
    {% block content %}{% endblock %}
  </body>
</html>

Child template:

{% extends 'base.html.twig' %}

{% block content %}
    <h1>Welcome {{ user.name }}</h1>
{% endblock %}

5. Includes & Partials

{% include 'partials/header.html.twig' %}

6. Escaping & Raw

{{ comment|e }}     {# Escapes HTML â†’ safe #}
{{ comment|raw }}   {# Output raw HTML (dangerous) #}

7. Translation (i18n)

{{ 'welcome.message'|trans({'%name%': user.name}) }}

8. Debugging

{{ dump(user) }}     {# Needs Symfony Debug enabled #}

9. Macros (Reusable Snippets)

{# forms.html.twig #}
{% macro input(name, value='') %}
  <input name="{{ name }}" value="{{ value }}">
{% endmacro %}

Usage:

{% import 'forms.html.twig' as forms %}
{{ forms.input('username') }}
