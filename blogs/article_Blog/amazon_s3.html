S3 organizes files into buckets. A bucket is like a folder at the top level of your S3 storage. Inside a bucket, you can have many objects (files), each identified by a unique key (the file name or path).

Each object in S3 has:

  • Optional metadata
  • The object data (file content itself)

üîê S3 Access: Public vs. Private

By default, S3 objects are private, meaning no one can access them directly from a URL. You have two main ways to make them available:

  • Public access ‚Äî make your bucket or specific files publicly readable.
  • Pre-signed URLs ‚Äî generate temporary, secure URLs that allow limited access.

ü™£ Uploading Files to S3

You can upload files through:

  • The AWS Management Console (drag and drop)
  • The AWS CLI
  • Programmatically via the AWS SDK (for JavaScript, Python, etc.)

Here‚Äôs an example using the AWS Management Console:

  1. Go to https://s3.console.aws.amazon.com/
  2. Click Create bucket and name it (e.g., my-image-bucket).
  3. Choose a region (e.g., us-east-1).
  4. Uncheck ‚ÄúBlock all public access‚Äù if you want to serve images publicly.
  5. Upload images using the Upload button.

üßæ Getting Images from S3

Now, let‚Äôs go step-by-step on how to get an image from an S3 bucket programmatically.

‚öôÔ∏è Get Image Using AWS SDK (Node.js Example)

If your bucket is private, you can use AWS SDK for JavaScript to fetch images securely.

npm install @aws-sdk/client-s3

Then use code like this:

import { S3Client, GetObjectCommand } from "@aws-sdk/client-s3";
import { getSignedUrl } from "@aws-sdk/s3-request-presigner";

const s3 = new S3Client({
  region: "us-east-1",
  credentials: {
    accessKeyId: process.env.AWS_ACCESS_KEY_ID,
    secretAccessKey: process.env.AWS_SECRET_ACCESS_KEY
  }
});

// Generate a pre-signed URL for an image
async function getImageUrl() {
  const command = new GetObjectCommand({
    Bucket: "my-image-bucket",
    Key: "images/profile.jpg"
  });

  const url = await getSignedUrl(s3, command, { expiresIn: 3600 }); // 1 hour
  console.log("Pre-signed URL:", url);
}

getImageUrl();

This will output a secure, temporary URL you can use in your app to fetch the image:

<img src="https://example-signed-url.amazonaws.com" alt="Profile Image" />
